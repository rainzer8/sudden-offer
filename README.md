# 📖 小林coding 镜像站

基于 [VitePress](https://vitepress.dev/) 构建的技术面试知识库。

原始站点：[xiaolincoding.com](https://xiaolincoding.com/)
GitHub 源码：[CS-Base](https://github.com/xiaolincoder/CS-Base)

## 本地运行

```bash
npm install
npm run dev      # 开发模式 http://localhost:5173
npm run build    # 构建
npm run preview  # 预览构建结果
```

## 部署

推送到 GitHub 后自动通过 GitHub Actions 部署到 GitHub Pages。
