## 🗺️ 进程虚拟地址空间布局

```
高地址
┌─────────────┐
│  内核空间     │
├─────────────┤
│  栈          │  ← 向下增长
│  ...        │
│  内存映射区   │
│  ...        │
│  堆          │  ← 向上增长
│  BSS 段      │
│  数据段       │
│  代码段       │
└─────────────┘
低地址
```
