## 💭 关于速通

大家好，我是小速 🤗。

曾经我也苦恼于求职面试中的各种难题——晦涩的知识点、无从下手的准备、让人紧张的面试现场。但在我啃了一本又一本的书，经历一次又一次的面试后，终于摸索出了一套属于自己的通关方法。

我想着，这世界上肯定还有很多朋友跟我有一样的苦恼。曾经淋过雨，现在想给你们撑把伞。

于是，速通offer 诞生了——从技术岗起步，逐步覆盖产品、运营、设计、销售等全岗位，让每一个求职者都能找到属于自己的面试通关攻略。

跟着速通，面试通关就这么简单！
